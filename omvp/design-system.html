Template Usage

Quick Start for Client Projects

1. Customize Brand Colors: Update the --template-primary, --template-secondary, and --template-accent CSS variables in the design system.

2. Apply Components: Use the pre-built components (buttons, cards, forms) in your HTML with consistent iOS styling.

3. Follow Typography: Use the iOS typography classes (.text-large-title, .text-body, etc.) for consistent text hierarchy.

4. Dark Mode Support: The system automatically supports dark mode based on user preference or manual toggle.
