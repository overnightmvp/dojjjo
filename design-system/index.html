Typography

Three sizes. System fonts. High contrast.

Large (32px) - Headings
Medium (20px) - Subheadings
Small (16px) - Body text and interface elements

Usage Rules:

  • Font weights: 400 (normal), 600 (bold), 700 (headings)
  • Never use font-weight: 900 (kills hierarchy)
  • Minimum 16px for body text (accessibility)
  • Line height: 1.2 (headings), 1.5 (body)

Usage Rules

Do this, not that. Keep it simple.

✅ DO

  • Use 16px minimum font size
  • High contrast colors (4.5:1 ratio)
  • Mobile-first design
  • Simple, clear language
  • Left-border accents for cards
  • 8px, 16px, 32px spacing

❌ DON'T

  • Font sizes smaller than 16px
  • Low contrast text
  • Complex shadows/effects
  • Too many colors
  • Heavy font weights (900)
  • Random spacing values
